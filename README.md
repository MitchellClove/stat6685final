 ### How to Run
 Each model is run separately, and the submissions.csv from each model is then combined in the hybrid model. Run each notebook (lstm.ipynb, catboost_04_ipynb.hybrid, cnn.ipynb) and then run the final Jupyter notebook to get the hybrid submission.csv.
 ### Data
 Data can be downloaded from https://www.kaggle.com/competitions/mallorn-astronomical-classification-challenge/data and the splits are stored in ./data/mallorn-astronomical-classification-challenge
